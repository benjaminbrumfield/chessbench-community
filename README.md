# chessbench-community

This repository is the community venue for [ChessBench](https://chessbench.ai) — a public benchmark for how well AI plays chess. It contains no code; the source lives elsewhere.

For requests, suggestions, and conversation, see [Discussions](https://github.com/benjaminbrumfield/chessbench-community/discussions).

## Categories

- **Model Requests** — suggest a model to add to the board.
- **Matchup Requests** — propose a head-to-head matchup you'd like to see.
- **Ideas** — feature suggestions for the site itself.
- **Q&A** — questions about methodology, ratings, or how something works.
- **General** — everything else: chess chat, related projects, papers, observations.
- **Announcements** — updates from the maintainer.

Before posting a request, please search existing threads — upvoting (👍) an existing request is more useful than a duplicate.
